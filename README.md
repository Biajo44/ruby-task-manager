# Ruby-Task-Manager

A **simple Command Line Interface (CLI) Task Manager** built in **Ruby**.  
This project lets you **add, list, complete, and delete tasks** directly from the terminal.  
Ideal for learning **Ruby, object-oriented programming (OOP), and basic CLI apps**.

---

## 📝 Features

- Add a new task
- List all tasks with their status (completed or not)
- Mark a task as completed
- Delete a task
- Interactive terminal menu

---

## 🧩 How it Works

The project uses **two main classes**:

1. **Task** – Represents a single task:
   - Attributes:
     - `title` → the name of the task
     - `completed` → boolean to track if the task is done
     
2. **TaskManager** – Manages the task list and menu:
   - Stores tasks in an array
   - Provides methods to:
     - Add tasks
     - List tasks
     - Complete tasks
     - Delete tasks
     - Display the interactive menu

The program uses **loops, arrays, conditionals, and methods** to manage tasks.
